# 🔊 ملفات الصوت المطلوبة - سكينة

## 📂 المكان الصحيح
```
Quran-main/
└── public/
    └── audio/
        ├── notification.mp3   ← صوت الإشعار الذكي
        └── salawat.mp3        ← صوت "صلِّ على النبي"
```

## 🎵 مواصفات الملفات

### `notification.mp3`
- **الوظيفة:** يُشغَّل مع كل إشعار ذكي (تذكير القرآن، نقاط، إلخ)
- **المدة المناسبة:** 1-3 ثواني
- **الصوت المقترح:** جرس هادئ أو نغمة إسلامية راقية

### `salawat.mp3`  
- **الوظيفة:** "اللهم صلِّ على محمد ﷺ" - يُشغَّل دورياً
- **المدة المناسبة:** 3-6 ثواني
- **الصوت المقترح:** تسجيل صوتي لـ "اللهم صلِّ على محمد" بصوت واضح

## ✅ كيف تختبر؟
1. افتح القائمة (☰) → اضغط "إعدادات"
2. في تبويب "الإشعارات" ← اضغط "اختبار الصوت"
3. في تبويب "صلِّ على النبي" ← اضغط "تجربة الصوت"
