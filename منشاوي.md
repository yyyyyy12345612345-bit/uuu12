## 🎨 ألوان الخلفية (3 أشرطة)

التصميم يتكون من 3 أشرطة أفقية:

```
┌──────────────────────┐
│   شريط أسود علوي      │  y: 0 → 380    (380px)  fill: #000000
├──────────────────────┤
│   شريط رمادي وسطي    │  y: 380 → 900  (520px)  fill: #383838
├──────────────────────┤
│   شريط أسود سفلي     │  y: 900 → 1280 (380px)  fill: #000000
└──────────────────────┘
```

### ألوان SVG (render.mjs)
```javascript
const bgRects = `
  <rect x="0" y="0"   width="720" height="380" fill="#000000" />
  <rect x="0" y="380" width="720" height="520" fill="#383838" />
  <rect x="0" y="900" width="720" height="380" fill="#000000" />
`;
```

### ألوان المعاينة (VideoPreview.tsx)
```tsx
// الخلفية الرئيسية: أسود
backgroundColor: "#000000"

// الشريط الرمادي الوسطي
<div className="absolute top-[29.69%] bottom-[29.69%] left-0 right-0 bg-[#383838] z-10" />
```

### خلفية Sharp (الرندرة النهائية)
```javascript
sharp({ create: { width: 720, height: 1280, channels: 3, background: { r: 0, g: 0, b: 0 } } })
```

## 🎯 قواعد التصميم العامة

1. **الصورة دائماً `object-contain` / `meet`** — لا نقص الصورة أبداً
2. **الحواف بلون بيج/رمادي خفيف** — `rgba(138, 128, 112, 0.4)` بسمك `3-4px`
3. **الأيقونات صغيرة ومتناسقة** — `scale(0.95)` في SVG، `15-16px` في المعاينة
4. **الخلفية 3 أشرطة** — أسود/رمادي/أسود
5. **النصوص بيضاء** — العنوان `bold #fff`، الاسم `50% opacity`
6. **المعاينة والسيرفر متطابقين** — أي تعديل يتم في الاثنين معاً
