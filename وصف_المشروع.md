# وصف مشروع تطبيق يقين | Yaqeen

## 📱 نظرة عامة على المشروع

تطبيق إسلامي متكامل مبني بتقنيات حديثة يوفر تجربة فريدة لقراءة القرآن الكريم، الأذكار، أوقات الصلاة، والمزيد من الميزات الروحية. التطبيق مصمم ليكون رفيقاً يومياً للمسلم في عباداته اليومية.

---

## 🎯 الأهداف الرئيسية

1. **تسهيل قراءة القرآن**: واجهة سهلة ومريحة لقراءة المصحف الشريف
2. **تنظيم الأذكار**: مكتبة شاملة لأذكار الصباح والمساء والنوم
3. **التحفيز والتشجيع**: نظام نقاط ولوحة شرف لتحفيز المستخدمين على المواظبة
4. **تحديد القبلة**: بوصلة إلكترونية دقيقة لتحديد اتجاه القبلة
5. **أوقات الصلاة**: عرض أوقات الصلاة مع إمكانية التنبيهات

---

## 🛠️ التقنيات المستخدمة

### Frontend
- **Next.js 16.2.3** - إطار عمل React مع App Router و Static Export
- **React 19.2.4** - مكتبة JavaScript لبناء واجهات المستخدم
- **TypeScript** - لغة برمجة قوية توفر التحقق من الأنواع
- **Tailwind CSS 4** - إطار عمل CSS للتصميم السريع والمتجاوب
- **Framer Motion** - مكتبة للرسوم المتحركة السلسة

### Backend & Database
- **Firebase Firestore** - قاعدة بيانات سحابية لتخزين بيانات المستخدمين والنقاط
- **Firebase Authentication** - نظام مصادقة آمن (Google Sign-In)

### Mobile
- **Capacitor** - إطار عمل لتحويل تطبيقات الويب إلى تطبيقات أندرويد أصلية
- **Android SDK** - لبناء APK النهائي

### المكتبات الإضافية
- **Lucide React** - أيقونات حديثة وخفيفة الوزن
- **Firebase SDK** - للتكامل مع خدمات Firebase
- **Capacitor Plugins**:
  - `@capacitor/app` - إدارة التطبيق
  - `@capacitor/geolocation` - تحديد الموقع الجغرافي
  - `@capacitor/local-notifications` - الإشعارات المحلية
  - `@capacitor/push-notifications` - الإشعارات الفورية
  - `@capawesome/capacitor-google-sign-in` - تسجيل الدخول عبر Google

## ✨ الميزات الرئيسية

### 1. 📖 المصحف الإلكتروني (Digital Mushaf)
- قراءة القرآن الكريم بخط واضح وجميل
- دعم جميع سور القرآن (114 سورة)
- إمكانية البحث عن آيات محددة
- تتبع التقدم في القراءة
- حفظ آخر موضع للقراءة
- تكبير وتصغير النص حسب الرغبة

### 2. 🕌 أوقات الصلاة (Prayer Times)
- عرض أوقات الصلاة الخمس بدقة عالية
- تحديد الموقع تلقائياً باستخدام GPS
- حساب أوقات الصلاة بناءً على الإحداثيات الجغرافية
- تنبيهات صوتية عند دخول وقت الصلاة
- دعم طرق حساب مختلفة (أم القرى، رابطة العالم الإسلامي، إلخ)

### 3. 🧭 البوصلة والقبلة (Qibla Compass)
- بوصلة إلكترونية دقيقة لتحديد اتجاه القبلة
- استخدام مستشعرات الجهاز للحصول على دقة عالية
- عرض المسافة إلى الكعبة المشرفة
- واجهة بصرية جميلة وسهلة الاستخدام

### 4. 📿 الأذكار والأدعية (Athkar)
#### أذكار الصباح
- مجموعة شاملة من أذكار الصباح
- عداد لكل ذكر لتتبع عدد التكرارات
- نقاط تشجيعية عند الانتهاء

#### أذكار المساء
- أذكار المساء الكاملة
- تتبع التقدم في كل ذكر
- نظام مكافآت للنشاط المستمر

#### أذكار النوم
- أذكار ما قبل النوم
- أدعية الاستيقاظ
- تسهيل المواظبة عليها

#### مكتبة الأذكار
- تصنيفات متعددة للأذكار
- بحث سريع في الأذكار
- إمكانية إضافة أذكار مفضلة

### 5. 🔢 العدادات الرقمية (Digital Counters)

#### تسبيح (Tasbeeh)
- تصميم عصري وجذاب
- اهتزاز خفيف عند الضغط (50ms)
- حماية من النقر السريع (Anti-spam: 200ms cooldown)
- نقاط: 3 نقاط لكل ضغطة
- حد يومي: 200 نقطة

#### استغفار (Istighfar)
- تصميم باللون الوردي (Rose-500)
- نص: "أَسْتَغْفِرُ اللَّهَ"
- اهتزاز خفيف (30ms)
- نقاط: 1 نقطة لكل ضغطة
- حد يومي: 1000 نقطة
- حماية من النقر السريع (150ms cooldown)

#### الصلاة على النبي (Salawat)
- تصميم باللون الأزرق (Blue-500)
- نص: "اللَّهُمَّ صَلِّ عَلَى مُحَمَّدٍ"
- اهتزاز خفيف (30ms)
- نقاط: 1 نقطة لكل ضغطة
- حد يومي: 1000 نقطة
- حماية من النقر السريع (150ms cooldown)

### 6. 🏆 نظام النقاط ولوحة الشرف (Points & Leaderboard)

#### أنواع النقاط:
- **نقاط القرآن (quranPoints)**: 100 نقطة كحد يومي
- **نقاط الأذكار (athkarPoints)**: 200 نقطة كحد يومي
- **نقاط الاستماع (listenPoints)**: 200 نقطة كحد يومي
- **نقاط الفيديو (videoPoints)**: 100 نقطة كحد يومي
- **نقاط المكافآت (bonus)**: 1000 نقطة كحد يومي
- **نقاط الاستغفار (istighfarPoints)**: 1000 نقطة كحد يومي
- **نقاط الصلاة على النبي (salawatPoints)**: 1000 نقطة كحد يومي

#### لوحة الشرف:
- ترتيب عام لجميع المستخدمين
- ترتيب حسب الدولة/المحافظة
- ترتيب حسب نوع النشاط (القرآن، الأذكار، الاستماع، الاستغفار، الصلاة على النبي)
- عرض أفضل 3 متسابقين بتصميم مميز (منصة التتويج)
- إحصائيات مفصلة لكل مستخدم
- تحديث فوري للترتيب

#### المهام اليومية (Quests):
- مهام يومية وأسبوعية
- مكافآت إضافية عند الإنجاز
- تتبع المهام المكتملة

### 7. 👤 الملف الشخصي (User Profile)
- تسجيل الدخول عبر Google
- اسم مميز (Username) بالإنجليزية
- اسم العرض (Display Name) بالعربية
- رقم الهاتف
- الدولة/المحافظة
- صورة شخصية
- إحصائيات شخصية
- تعديل البيانات

### 8. 🎥 إنشاء الفيديوهات (Video Creation)
- إنشاء فيديوهات إسلامية احترافية
- قوالب جاهزة للاستخدام
- إضافة نصوص وآيات قرآنية
- تصدير بجودة عالية

### 9. 💬 المساعد الذكي (ChatBot)
- مساعد ذكي للإجابة على الأسئلة الإسلامية
- معلومات عن القرآن والسنة
- تفسير الآيات

### 10. ⚙️ الإعدادات (Settings)
- تخصيص طريقة الحساب لأوقات الصلاة
- اختيار المؤذن المفضل
- تفعيل/تعطيل الإشعارات
- تغيير الثيم (فاتح/داكن)
- إعدادات اللغة

## 📊 هيكل قاعدة البيانات (Firestore)

### مجموعة المستخدمين (users collection)
```typescript
{
  uid: string,                    // معرف المستخدم الفريد
  username: string,               // الاسم المميز (إنجليزي)
  displayName: string,            // اسم العرض (عربي)
  email: string,                  // البريد الإلكتروني
  photoURL: string,               // رابط الصورة الشخصية
  phoneNumber: string,            // رقم الهاتف
  governorate: string,            // المحافظة/الدولة
  totalPoints: number,            // إجمالي النقاط
  quranPoints: number,            // نقاط القرآن
  athkarPoints: number,           // نقاط الأذكار
  listenPoints: number,           // نقاط الاستماع
  videoPoints: number,            // نقاط الفيديو
  istighfarPoints: number,        // نقاط الاستغفار
  salawatPoints: number,          // نقاط الصلاة على النبي
  streakDays: number,             // أيام الاستمرارية
  badges: string[],               // الشارات المحققة
  readAyahs: number,              // عدد الآيات المقروءة
  audioSeconds: number,           // ثواني الاستماع
  videoRendersCount: number,      // عدد الفيديوهات المنشأة
  isBanned: boolean,              // هل المستخدم محظور
  createdAt: string,              // تاريخ الإنشاء
  lastActive: string              // آخر نشاط
}
```

### الإحصائيات اليومية (users/{uid}/stats/{date})
```typescript
{
  points: number,                 // النقاط المجمعة اليوم
  lastUpdate: timestamp           // آخر تحديث
}
```

### المهام المكتملة (users/{uid}/completed_quests/{questId})
```typescript
{
  completedAt: timestamp,         // تاريخ الإكمال
  points: number                  // النقاط المكتسبة
}
```

### السور المكتملة (users/{uid}/completed_surahs/{surahId})
```typescript
{
  surahId: number,                // رقم السورة
  completedAt: timestamp,         // تاريخ الإكمال
  points: number                  // النقاط المكتسبة
}
```

### المهام العالمية (global_quests)
```typescript
{
  id: string,                     // معرف المهمة
  title: string,                  // عنوان المهمة
  type: string,                   // نوع المهمة (quran, athkar, daily)
  target: number,                 // الهدف
  points: number,                 // النقاط
  createdAt: timestamp            // تاريخ الإنشاء
}
```
