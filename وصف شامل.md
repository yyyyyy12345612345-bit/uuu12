## 🏗️ بنية الموقع (Architecture)

### هيكل الصفحات (Single-Page Application)
الموقع يستخدم **Catch-All Route** واحد (`[[...slug]]/page.tsx`) بيحمّل كل الصفحات client-side عن طريق `CatchAllPageClient.tsx`. مفيش ملف HTML منفصل لكل صفحة — الكل بيتحمّل في SPA واحدة سريعة.

### نظام التنقل
- **Custom Navigation System** (`src/lib/navigation.ts`): بيستخدم custom events عشان يتجاوز Next.js routing العادي — وده بيخلّي التنقل بين الصفحات **فوري** من غير إعادة تحميل
- **Bottom Navigation Bar** (`Navigation.tsx`): 6 tabs (مصحف، يوميات، رتب، مكتبة، صلاة، فيديو) مع indicator ذهبي
- **Global Menu** (`GlobalMenu.tsx`): menu جانبي منزلق فيه كل الأقسام والتفاعلات (شير، تحميل APK، دليل النقاط)

### أنظمة الرندر (Render):

#### 1. الرندر المحلي (Browser-based)
- **التقنية**: Canvas 2D + AudioContext + MediaRecorder
- **الخطوات**: يرسم كل فريم على `<canvas>` (720×1280) ← يلتقط الـ stream ← يسجل WebM
- **المميزات**: Audio Visualizer حي، فلاتر CSS محاكاة، تأثيرات overlay
- **السرعة**: أبطأ (بيعتمد على جهاز المستخدم)

#### 2. الرندر عبر السيرفر (Hyper Render) — الملف `render.mjs`
- **التقنية**: Sharp (SVG compositing) + FFmpeg (video encoding)
- **الميزة**: **Zero Chromium** — لا تسريب ذاكرة
- **السيرفر**: Express Server على Hugging Face Space
- **الخطوات**: يحمّل الفونتات (6 خطوط من Google Fonts) ← يحمّل الخلفية (صورة/فيديو) ← يحمّل مقاطع الصوت لكل آية ← يقيس المدة بـ ffprobe ← يركّب SVG لكل سطر ← يدمج الصوت بـ FFmpeg ← يخرج MP4
- **API Endpoint**: POST → `{ jobId }` ← GET `/status/:jobId` ← GET `/download/:filename`
- **دعم خلفيات الفيديو**: overlay شفاف للـ PNG frames على الـ video background

#### 3. الرندر عبر Remotion (ملف `final-server.mjs`)
- قديم — بيستخدم Chromium عشان يـ render React components لفيديو

### الملف المحلي ML (`ml-model.ts` — 1109 سطر):
- **نموذج Vector Space Model** حقيقي
- **TF-IDF**: Term Frequency-Inverse Document Frequency
- **Cosine Similarity**: حساب التشابه بين سؤال المستخدم و 50+ وثيقة تدريبية
- **Stop Words**: فلترة كلمات عربية شائعة
- **70+ تصنيف**: ترحيب، مطور، موقع، نقاط، فيديو، صلاة، مصحف، أذكار، الخ
- **السرعة**: < 1 مللي ثانية

### قدرات الشات بوت:
- إجابة أسئلة دينية (تفسير، أحكام، فتاوى)
- إرشاد داخل الموقع (فين صفحة كذا)
- عرض النقاط والترتيب
- تحديث الملف الشخصي (الاسم، البلد، الهاتف)
- إنشاء خطة حفظ قرآنية
- توليد مسابقات دينية
- كشف الإهانات والرد بلطف
- دمج لوحة الشرف في السياق
- رفض الأسئلة السياسية وخارج النطاق
